## backoff algorithm set
```conf
multiplier = 2
base = 500msmax 
wait_interval = 8s
max-retry = 6
```

## Run Server
```sh
cd server

cd build

./script.sh
```

## Run Client
```sh
cd client

cd build

./script.sh
```
